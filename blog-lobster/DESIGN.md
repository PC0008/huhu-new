# 龙虾风格个人博客设计文档

## 配色方案

### 主色调
- **背景色**: #FDF6F0 (米色/奶油色)
- **强调色**: #E85A4F (珊瑚红)
- **文字色**: #2D2D2D (深灰黑)
- **次要文字**: #666666 (中灰)
- **辅助色**: #F5E6D3 (浅米色)

### 边框/阴影
- **卡片边框**: rgba(0,0,0,0.05)
- **卡片阴影**: 0 2px 8px rgba(0,0,0,0.08)
- **悬浮阴影**: 0 4px 16px rgba(0,0,0,0.12)

## 页面结构

### 1. 首页 (Home)
- Hero区: 大图 + 标题 + CTA按钮
- 最新日记预览: 瀑布流卡片
- 技能展示区: 3个特性卡片
- 留言板区

### 2. 日记页 (Diary)
- 瀑布流时间轴卡片
- 每篇日记有日期标签
- 卡片包含: 图片、标题、摘要、日期

### 3. 文章页 (Articles)
- 左侧: 文章列表 (标题、摘要、标签、日期)
- 右侧: 热门排行侧边栏 + 分类统计

### 4. 技能页 (Skills)
- 技能包展示 (大卡片)
- 全部技能网格展示

## 组件规范

### 卡片
- 圆角: 12px
- 内边距: 16px-24px
- 背景: 白色
- 阴影: 柔和

### 按钮
- 主按钮: 珊瑚红背景 + 白色文字
- 次按钮: 透明背景 + 珊瑚红边框
- 圆角: 8px
- 内边距: 8px 16px

### 标签
- 圆角: 16px (pill形状)
- 背景: 浅米色
- 文字: 珊瑚红

## 响应式断点
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
